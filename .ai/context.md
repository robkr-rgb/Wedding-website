# Context — Wedding Website

Single-page static wedding website. No framework, no build step.

## Stack

- `src/index.html` — full site (nav, hero, story, details, gallery, RSVP, footer)
- `src/server.js` — minimal Node.js static file server (port 3001)
- `src/photos/` — drop photos here; reference via `data-src` on `.gallery-item` elements
- Formspree form ID: `mojzkngo`

## Design

Gold/cream/dark palette. Georgia serif. Mobile-responsive. Live countdown in hero.

## Current placeholders

- Our Story text (hero section)
- Dress code details
- Accommodation info
- 6 gallery photo slots (`data-src` attributes on `.gallery-item` divs)
- Captions on gallery items 1 and 6
